HOW TO RUN
==========

OPTION 1: Docker (Recommended) 🐳
  docker-compose up
  → Frontend: http://localhost:3000
  → Backend: http://localhost:8000
  → API Docs: http://localhost:8000/docs

OPTION 2: Manual Setup
  
  Terminal 1 - Backend:
    cd backend
    python -m venv venv
    venv\Scripts\activate  # Windows
    pip install -r requirements.txt
    uvicorn app.main:app --reload

  Terminal 2 - Frontend:
    cd frontend
    python -m http.server 3000

OPTION 3: Using Scripts
  
  Windows:
    run-backend.bat  # Terminal 1
    run-frontend.bat # Terminal 2

  macOS/Linux:
    chmod +x run-backend.sh run-frontend.sh
    ./run-backend.sh  # Terminal 1
    ./run-frontend.sh # Terminal 2

FIRST STEPS
===========

1. Sign Up
   Click "Sign Up" in navbar and create an account

2. Report an Issue
   Click "Report Issue" and fill in the form with:
   • Issue title and description
   • Category (Pothole, Water Leak, Streetlight, etc.)
   • Priority (Low, Medium, High, Critical)
   • Location (auto-detect or manual)

3. Check Dashboard
   Navigate to "Dashboard" to see:
   • Real-time statistics
   • Issues by category
   • Trending issues
   • Recent reports

4. View Leaderboard
   Check "Leaderboard" to:
   • See community leaders
   • View reputation scores
   • Track badges earned

5. Verify Issues
   Help verify other users' reports and earn reputation points!

TROUBLESHOOTING
===============

Port Already in Use:
  → Change port: uvicorn app.main:app --port 8001

Dependencies Won't Install:
  → Use Python 3.11+ (not 3.13 which has Rust build issues)
  → Or use Docker: docker-compose up

CORS Errors:
  → Ensure backend runs on http://localhost:8000
  → Check browser console for details

More Help:
